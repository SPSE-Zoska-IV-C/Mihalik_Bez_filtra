Bez Filtra
Webowa aplikacia postavena na Frameworku flasku ktorá použiva RSS feedy ktore sú parsovane cez Feedparser a očistené cez Beutifullsoup. Nasledne su spracované članky 
zgroupovane cez Jaccardov koeficient a nasledne gemini api tvori prehľadné body.
Aplikaciu viete zclonovať cez github, ďalej vám už len stačí vytvoriť virtualne prostredie a stiahnuť requirements.
Projekt je open source je dovoleneho ho využiť pokial poskytnete credit
